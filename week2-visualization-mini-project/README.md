# Week 2 – Data Visualization Mini Project

## Project Title
Data Visualization on Amazon E-commerce Sales Dataset

## Objective
To perform data visualization using Python and Matplotlib and create four different types of charts based on a real-world dataset.

## Dataset Used
- Source: Kaggle – Amazon Sale Report  
- File Used: Amazon Sale Report.csv  
- Contains transaction-level e-commerce sales data

## Tools and Libraries
- Python  
- Pandas  
- Matplotlib  
- Jupyter Notebook  

## Charts Created

### 1. Line Chart – Daily Sales Trend
- Shows total revenue generated on each date  
- Helps analyze sales growth over time  

### 2. Bar Chart – Sales by Category
- Compares total sales across different product categories  
- Identifies top-performing categories  

### 3. Scatter Plot – Quantity vs Amount
- Analyzes relationship between quantity sold and sales amount  

### 4. Pie Chart – Order Status Distribution
- Displays proportion of shipped, cancelled, and delivered orders
